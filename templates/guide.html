Welcome to My Novel List!

The following will be a guide to use this web application!


Home Page

There are four main buttons in the home page.

Add Filter: If you click this button the following window will popup.

This is where you can search your list with the following filters.

----- Rating -----
Enter any rating and it will filter by those ratings

----- Tags -----
You can enter any number of tags and it will filter the list by the results.

The format is: magic, high fantasy, novel
----- Review -----
You can enter any words, sentences or even incomplete sentences, it will find results that have the phrase in it somewhere.

After clicking confirm, you have to click search.

Note: if you have more than one filter, it will give you any results that matches any one filter, but the result is sorted by the best matches.


Clear filter: If you click this button, it will clear your filters and give you the full list.

Search bar: You can type any name in here and it you find you the results that matches that name

Search: If you click this button, it will search the list with your filters and the name in the search bar, if there is nothing in the search bar, it will ignore it and vise versa with the filters.

Adding books to your list

Add book: When you click this button, it will bring you to the following page.

Search: Same thing as the home page but except also you can search for books using the author name.

Add to list: When you click this button, the following page will popup.

Delete: Deletes the item off your list. (This page will appear again when you want to edit these values after you enter the book into your list)

Note: For the tags, if you want to enter a tag with more than one word,
you have to use "_" to separate the phrase, e.g. magic_fantasy

Now, back to the other page!

Not here?: If you click this button, the following page will show.

When you don't see your book in the database, you can add it manually.
After you click submit, it will be entered into the database for you and the other people using this web application to access as well.

Note: For the tags, it is the same deal as last time, use "_" as spaces.

Now, back to the other page!

Edit: If you click this button, the following page will show.

This is where you can edit the books in the database, (description, author, title).
Note: For the tags, it is the same deal as last time, use "_" as spaces.
CAUTION: When you delete books in here, it will delete it off the database,
which will automatically delete it off of every list that the book is on.

Editing books in your list

When you have a book in your list, it will look like the following.

Edit: When you click this button it will bring you to this page.

Looks familiar?
It is the same page as when you add a book to your list, but now you can edit the details of the book or delete the book off your list.
That is it! You can always come back to this page by clicking "guide" in the nav bar.
